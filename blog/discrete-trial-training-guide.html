Discrete Trial Training (DTT): A Complete Guide for RBTs

Discrete Trial Training (DTT) is one of the most widely used teaching strategies in applied behavior analysis (ABA). Originally developed as part of the Lovaas Method in the 1960s, DTT breaks complex skills into small, teachable units and uses structured repetition and reinforcement to build mastery.

As an RBT, you'll likely implement DTT daily. This guide covers everything you need to know.

What Is DTT?

Discrete Trial Training is a structured teaching method that uses a series of individual learning opportunities (trials) to teach specific skills. Each trial is "discrete" ‚Äî it has a clear beginning, middle, and end.

DTT is particularly effective for teaching:

  • Receptive and expressive language (labeling, following instructions)
  • Academic skills (letters, numbers, colors, shapes)
  • Imitation skills
  • Play skills
  • Social skills (greetings, turn-taking)
  • Daily living skills

Step-by-Step Example: Teaching Color Identification

Here's how a DTT sequence might look when teaching a child to identify the color red:

Trial 1 (Prompted)

  1. SD: RBT places a red card and a blue card on the table. "Touch red."
  2. Prompt: RBT points to the red card (gestural prompt)
  3. Response: Child touches the red card
  4. Consequence: "Awesome! That's red!" + provides a small treat
  5. ITI: RBT pauses for 3 seconds, rearranges cards

Trial 2 (Fading Prompt)

  1. SD: "Touch red."
  2. Prompt: RBT glances toward the red card (reduced prompt)
  3. Response: Child touches the red card
  4. Consequence: "Yes! Red!" + high five
  5. ITI: 3-second pause

Trial 3 (Independent)

  1. SD: "Touch red."
  2. Prompt: None
  3. Response: Child independently touches the red card
  4. Consequence: "Amazing job! You found red all by yourself!" + preferred item
  5. ITI: 3-second pause

Prompting in DTT

Prompts are temporary assistance to help the learner respond correctly. The goal is always to fade prompts as quickly as possible so the learner responds independently.

Prompt Hierarchy (Most to Least)

  1. Full physical: Hand-over-hand guidance
  2. Partial physical: Light touch or nudge
  3. Model: Demonstrating the correct response
  4. Verbal: Telling the answer or giving a hint
  5. Gestural: Pointing, nodding, or looking at the correct answer
  6. Positional: Placing the correct item closer to the learner
  7. Independent: No prompt (the goal)

Prompt Fading Strategies

  • Most-to-Least (MTL): Start with the most intrusive prompt and gradually fade to less intrusive ones. Best for learners who need more support initially.
  • Least-to-Most (LTM): Start with no prompt and increase assistance only if the learner doesn't respond correctly. Promotes more independent responding.
  • Time Delay: Insert a time delay between the SD and the prompt, gradually increasing the delay to give the learner time to respond independently.

Error Correction Procedures

When a learner responds incorrectly, RBTs use error correction to help them learn the correct response. A common procedure:

  1. Provide feedback: Brief, neutral response (e.g., "Let's try again" or simply "No")
  2. Re-present the SD: Give the instruction again
  3. Prompt the correct response: Use whatever prompt level ensures a correct response
  4. Provide differential reinforcement: Reinforce the prompted correct response, but with less enthusiasm than an independent correct response
  5. Run a transfer trial: Present the SD again with a faded prompt to test for independent responding

Data Collection in DTT

Data collection is essential in DTT. For each trial, RBTs typically record:

  • Correct (+) or Incorrect (‚àí): Whether the response was correct
  • Prompt level used: What type of prompt was needed (or independent)
  • Date and session information

Common data formats include:

  • Trial-by-trial data: Record the outcome of every single trial
  • First trial data: Record only the first response in each session (to measure true independent performance)
  • Percentage correct: Number of correct trials √∑ total trials √ó 100

Mastery Criteria

A skill is considered "mastered" when the learner meets predetermined criteria, such as:

  • 80% or higher accuracy across 2‚Äì3 consecutive sessions
  • Independent responding (no prompts needed)
  • Across different people, settings, or materials (generalization)

The specific mastery criteria are set by the supervising BCBA in the treatment plan.

Tips for Effective DTT

  1. Keep it fun: Use enthusiastic praise, vary reinforcers, and incorporate the learner's interests
  2. Be consistent: Present the SD the same way each time until the skill is acquired
  3. Reinforce immediately: Deliver reinforcement within 1‚Äì2 seconds of the correct response
  4. Fade prompts systematically: Don't get stuck at a high prompt level ‚Äî follow the prompt fading plan
  5. Mix mastered and new targets: Intersperse easy, mastered items with new learning targets to maintain motivation
  6. Take accurate data: Record data during or immediately after each trial ‚Äî don't try to remember later
  7. Follow the BCBA's plan: Always implement DTT programs as written by the supervising BCBA
  8. Stay organized: Have all materials ready before starting a DTT session

Frequently Asked Questions

What is discrete trial training (DTT)?

Discrete Trial Training (DTT) is a structured ABA teaching method that breaks skills into small, isolated components and teaches them through repeated practice with clear antecedents, responses, and consequences. Each trial has five components: discriminative stimulus (SD), prompt (if needed), response, consequence, and inter-trial interval.

What are the 5 components of a discrete trial?

The five components are: (1) Discriminative Stimulus (SD) ‚Äî the instruction, (2) Prompt ‚Äî assistance if needed, (3) Response ‚Äî the learner's answer, (4) Consequence ‚Äî reinforcement or error correction, and (5) Inter-trial Interval (ITI) ‚Äî a brief pause between trials.

What is the difference between DTT and NET in ABA?

DTT is a structured, therapist-directed teaching method conducted in a controlled setting with massed trials. NET (Natural Environment Teaching) is a more naturalistic approach that follows the learner's motivation and takes place during everyday activities. Most ABA programs use both methods.
